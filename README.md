# Placement Management Portal

A Full Stack Placement Management Portal built using HTML, CSS, JavaScript, Spring Boot, and MySQL.

## Features

* Student Registration
* Company Registration
* Job Posting Management
* Login & Signup System
* Skill-Based Job Matching
* View Relevant Jobs Based on Student Skills
* Responsive User Interface

## Technologies Used

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Java
* Spring Boot
* REST API

### Database

* MySQL

## Project Flow

1. Student registers profile
2. Company posts jobs
3. Job postings are stored in database
4. Student skills are matched with job requirements
5. Relevant jobs are displayed after registration
